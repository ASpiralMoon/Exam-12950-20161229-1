#	EXAM

##	1.说明

一个基于spring+mybatis的控制台系统

##	2.功能

* 用户登录
* 未登录无法进入系统,自动跳转到登录页面
* 主页显示所有电影信息
* 单条删除电影
* 单条新增电影
* 单条编辑电影
* more


##	3.开发环境

* spring
* mybatis
* druid
* MySql
* aspectj
* more
